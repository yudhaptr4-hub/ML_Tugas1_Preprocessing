# ML Tugas 1 - Data Preprocessing

## Deskripsi
Proyek ini merupakan tugas preprocessing data mahasiswa yang mencakup:
- Handling missing values
- Normalisasi format tanggal
- Encoding data kategorikal
- Split data training dan testing
- Visualisasi data

## Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Dataset berisi data mahasiswa dengan beberapa permasalahan seperti missing values dan inkonsistensi format.

## Hasil
- Data berhasil dibersihkan dari missing values
- Format tanggal berhasil dinormalisasi
- Data kategorikal berhasil di-encode
- Dataset siap digunakan untuk machine learning
